# 固件更新指南

本文档指导您将固件更新到 ESP32-S3 ESM_POWER_SYSTEM。

## 固件文件

| 文件名 | 说明 |
|--------|------|
| `esm_power_firmware_{commit}_{date}.bin` | 正式版固件 |

> **说明**：文件名中的 `{commit}` 为版本标识，`{date}` 为编译日期。

## 准备工作

### 所需材料

- ESP32-S3 ESM_POWER_SYSTEM
- USB 数据线（Type-C 或 Micro-USB，取决于设备接口）
- 电脑（Windows 系统）

### 下载刷机工具

提供两种刷机工具供选择：

#### 方式一：Web 在线刷机（推荐）

无需安装任何软件，使用浏览器直接刷机。

- 工具地址：https://espressif.github.io/esptool-js/
- 要求：Chrome 89+ 或 Edge 89+ 浏览器

#### 方式二：桌面版刷机工具

下载乐鑫官方刷机工具 **ESP Flash Download Tool**：

- 下载地址：https://www.espressif.com/en/support/download/other-tools
- 选择 **Flash Download Tools** 下载
- 解压后运行 `flash_download_tool_x.x.x.exe`

## 刷机模式说明

根据不同的使用场景，选择合适的刷机模式：

| 模式 | 地址 | 说明 | 适用场景 |
|------|------|------|----------|
| **全量刷机** | 0x0 | 擦除整个 Flash 并写入固件 | 首次刷机、系统损坏、需要完全重置 |
| **App 分区刷机** | 0x10000 | 只刷写应用分区，保留配置数据 | 日常固件更新 |

> **重要提示**：
> - **全量刷机**会清除所有配置数据（包括设备设置、校准参数等），刷机后需要重新配置
> - **App 分区刷机**保留设备配置，推荐用于日常更新

## 刷机步骤

### 方式一：Web 在线刷机

#### 第一步：进入刷机模式

如果设备无法自动进入刷机模式，请手动操作：

1. 按住设备上的 **BOOT** 按钮
2. 短按 **RESET** 按钮
3. 松开 **BOOT** 按钮

#### 第二步：打开刷机页面

1. 使用 Chrome 或 Edge 浏览器访问：https://espressif.github.io/esptool-js/
2. 点击页面上的 **Connect** 按钮

#### 第三步：选择设备

1. 在弹出的串口选择窗口中，选择您的设备
2. 点击 **连接** 按钮

#### 第四步：配置固件

1. 在 **Flash Address** 栏输入：
   - **全量刷机**：输入 `0x0`
   - **App 分区刷机**：输入 `0x10000`
2. 点击 **Choose File** 按钮选择固件文件
3. 确认文件名和地址显示正确

#### 第五步：开始刷机

1. 点击 **Program** 按钮开始刷机
2. 等待进度条完成
3. 显示 **Done** 表示刷机成功

#### 第六步：重启设备

1. 刷机完成后，点击 **Reset** 按钮重启设备
2. 或者短按设备上的 **RESET** 按钮

---

### 方式二：桌面版刷机工具

#### 第一步：连接设备

1. 使用 USB 线将设备连接到电脑
2. 等待电脑识别设备（会自动安装驱动）

#### 第二步：进入刷机模式

如果设备无法自动进入刷机模式，请手动操作：

1. 按住设备上的 **BOOT** 按钮
2. 短按 **RESET** 按钮
3. 松开 **BOOT** 按钮

#### 第三步：配置刷机工具

1. 打开 ESP Flash Download Tool
2. 选择芯片类型：**ESP32-S3**
3. 选择串口：在下拉菜单中选择对应的 COM 口
4. 设置波特率：**460800**（默认值即可）
5. 点击 **...** 按钮选择固件文件
6. 在地址栏输入：
   - **全量刷机**：输入 `0x0`
   - **App 分区刷机**：输入 `0x10000`

#### 第四步：开始刷机

1. 点击 **START** 按钮
2. 等待进度条完成
3. 显示 **FINISH** 表示刷机成功

#### 第五步：重启设备

1. 刷机完成后，设备会自动重启
2. 如果没有自动重启，短按 **RESET** 按钮

## 验证刷机

设备重启后，请验证以下内容：

1. **启动动画**：屏幕显示启动动画
2. **主界面**：动画结束后显示主界面，包含温度、转速、功率等信息
3. **按键响应**：
   - 短按确认键：切换显示模式
   - 长按确认键：进入设置页面
   - 上下键：移动选择

## 常见问题

### 如何选择刷机模式？

- **首次刷机**：使用全量刷机（地址 0x0）
- **固件更新**：使用 App 分区刷机（地址 0x10000），保留设备配置
- **系统异常**：使用全量刷机（地址 0x0），完全重置设备

### 电脑无法识别设备

**现象**：设备管理器中没有显示 COM 口。

**解决方案**：
1. 更换 USB 数据线（部分线仅支持充电，不支持数据传输）
2. 安装 USB 转串口驱动：
   - 下载地址：https://www.silabs.com/developers/usb-to-uart-bridge-vcp-drivers
3. 重新插拔 USB 线

### 刷机失败

**现象**：刷机工具报错或进度条卡住。

**解决方案**：
1. 确保设备已进入刷机模式（按住 BOOT + 短按 RESET）
2. 降低波特率：将 460800 改为 115200
3. 关闭其他可能占用串口的程序
4. 重新插拔 USB 线

### 刷机后无显示

**现象**：刷机成功但屏幕无任何显示。

**解决方案**：
1. 短按 RESET 按钮重启设备
2. 检查电源供应是否正常
3. 等待 10-15 秒（首次启动可能较慢）

### 设备反复重启

**现象**：设备不断重启，无法正常启动。

**解决方案**：
1. 重新刷机，确保使用正确的固件文件
2. 检查固件文件是否完整（未损坏）
3. 尝试使用更低的波特率刷机

## 注意事项

### 通用注意事项

- 刷机过程中请勿断开 USB 连接
- 刷机前请确保设备电量充足或已连接电源
- 如遇问题，请勿频繁尝试，等待 30 秒后再试
- 刷机不会清除设备配置，但建议记录重要设置

### Web 在线刷机注意事项

- 仅支持 Chrome 89+ 或 Edge 89+ 浏览器，不支持 Firefox 和 Safari
- 需要使用 HTTPS 连接（网站已自动配置）
- 首次使用时，浏览器会请求串口访问权限，请点击"允许"
- 如果连接失败，请检查：
  - 浏览器版本是否符合要求
  - 是否有其他程序占用串口
  - USB 线是否支持数据传输

### 桌面版刷机工具注意事项

- 如果遇到驱动问题，请安装 USB 转串口驱动
- 刷机工具可能被杀毒软件拦截，请添加信任
